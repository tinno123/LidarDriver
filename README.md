# Lidar.py — LD系列激光雷达点云可视化

基于 Python 的 乐动品牌串口激光雷达点云显示程序，使用 OpenCV 进行可视化，采用面向对象封装和线程安全队列。可用于ros2开发、激光点云与相机数据标定实现简单的目标测距。

---

## 数据包解码格式

### 数据包结构

| 偏移 | 字段 | 大小 | 说明 |
|---|---|---|---|
| 0~1 | 包头 | 2B | 固定值 `0xAA 0x55` |
| 2 | CT | 1B | 包类型，bit0=1 表示新一圈开始 |
| 3 | LSN | 1B | 本包数据点数 |
| 4~5 | FSA | 2B | 起始角度（小端序） |
| 6~7 | LSA | 2B | 结束角度（小端序） |
| 8~ | Data | N×3B | 每个点 `[距离低8位, 距离高8位, 信号强度]` |

### 角度解码

```
原始值 = (MSB << 8) \| LSB     // 小端序
角度值 = (原始值 >> 1) / 64    // 单位：度
```

- 0° 为正上方（北向），角度顺时针递增
- 范围：0° ~ 359.99°

### 中间点角度插值

```
角度_i = ((LSA - FSA) % 360) / (LSN - 1) * i + FSA
```

`% 360` 用于处理扫描跨越 0° 边界的情况。

### 距离解码

每个数据点 3 字节：`[d_low, d_high, intensity]`

```
距离(mm) = ((d_high << 8) \| d_low) / 4.0
```

返回值为毫米（mm）。

### 极坐标 → 直角坐标变换

```
弧度 = radians((角度 - 90) % 360)
x = -sin(弧度) × 距离
y =  cos(弧度) × 距离
```

- 减去 90° 将极坐标 0°（北）对齐到屏幕 Y 轴向上
- 坐标系：水平向右为 X+，垂直向上为 Y+

### 屏幕映射

```
sx = int(x × scale + cx)    // cx = 图像宽度/2 = 320
sy = int(y × scale + cy)    // cy = 图像高度/2 = 320
```

- `scale = 0.1`：将毫米坐标缩放至像素坐标
- 图像尺寸：640 × 640

## 使用方法

### 硬件连接

1. 将 LD 系列激光雷达通过 USB 转串口模块连接至电脑
2. 确认设备端口号（默认为 `COM5`）

### 安装依赖

```powershell
pip install opencv-python numpy pyserial
```

### 运行

```powershell
python Lidar.py
```

### 操作

| 按键 | 功能 |
|---|---|
| `Q` | 退出程序 |

## 程序架构

```
                    ┌─────────────────┐
                    │   主线程 main()  │
                    │  OpenCV 显示循环 │
                    └──────┬──────────┘
                           │ GetData()
                    ┌──────▼──────────┐
                    │  LidarDriver    │
                    │  数据管理与坐标  │
                    │  变换/屏幕映射  │
                    └──────┬──────────┘
                           │ 队列传递
                    ┌──────▼──────────┐
                    │  读取线程       │
                    │  ReadData()     │
                    │  串口读取+解码  │
                    └─────────────────┘
```

### 双缓冲机制

- `CPoint`：当前累积的新数据点
- `LPoint`：上一帧完整数据的备份
- 当 `CPoint` 超过 400 个点时，视为攒够客观的显示数量，将其映射到屏幕坐标后返回显示，同时备份到 `LPoint`
- 不足 400 点时，返回上一帧的 `LPoint` 保证画面不空白,可用于保证当前while帧有足够的点云。

### 关键方法

| 方法 | 功能 |
|---|---|
| `DecodeAngle()` | 解析起始/结束角度 |
| `DecodeDistance()` | 解析数据点距离 |
| `DecodeMediumAngle()` | 线性插值计算中间点角度 |
| `projection()` | 极坐标→直角坐标变换 |
| `ReadData()` | 串口读取线程，持续解析数据包 |
| `GetData()` | 从队列取数据，做屏幕映射后返回 |
